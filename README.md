# Versatile WordPress Theme

A professional, modern WordPress theme with a clean and organized structure.

## Features

- Responsive design
- WooCommerce integration
- Custom post types support
- Block editor ready
- SEO optimized
- Translation ready

## Installation

1. Upload the theme to your `/wp-content/themes/` directory
2. Activate the theme through the WordPress admin
3. Configure theme settings in Appearance > Customize

## Development

This theme uses modern development tools:

```bash
# Install dependencies
npm install

# Build assets
npm run build

# Watch for changes
npm run watch
```
